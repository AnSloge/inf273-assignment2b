# INF273 Metaheuristics

Course workspace organized by assignment with separate source code, data, and results folders.

## Structure

- `assignments/assignment2/src`: Assignment 2 source code.
- `assignments/assignment2/data`: Assignment 2 input instances.
- `assignments/assignment2/results`: Assignment 2 generated outputs and saved solutions.
- `assignments/assignment3/src`: Assignment 3 source code.
- `assignments/assignment3/data`: Assignment 3 input instances.
- `assignments/assignment3/results`: Assignment 3 generated outputs and saved solutions.
- `assignments/assignment4/src`: Assignment 4 source code.
- `assignments/assignment4/data`: Assignment 4 input instances.
- `assignments/assignment4/results`: Assignment 4 generated outputs.
- `common/`: Shared utilities reused across assignments.

## Running

Run scripts from the repository root so imports resolve consistently.

Examples:

```bash
python assignments/assignment2/src/brs.py
python assignments/assignment3/src/local_search.py
python assignments/assignment3/src/simulated_annealing.py
```
